# MyTrajets
Une entreprise de transport routier dispose d’une flotte de camions et de remorques utilisée pour le transport de marchandises entre différents sites (entrepôts, clients, fournisseurs, ports, etc.). Aujourd’hui, la gestion de cette flotte est réalisée de manière essentiellement manuelle (fichiers Excel, appels téléphoniques, documents papier)
